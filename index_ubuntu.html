Overview

PHP is a general use programming language that has been an integral part of web development since around the time of its creation in 1994.

The Standard PHP Library (SPL) is a collection of interfaces and classes that are meant to solve common problems. Here are some of the ways it can be useful:

  • Performing various operations on strings such as finding the length, substituting, and searching for substrings
  • Working with Date and Time objects and formatting them

    • Let's get a basic feel for PHP's syntax and standard library with some code:

                function say_hello( $name, $date ) {
              echo "Hello, " . $name . "\n";
              echo "Today is " . $date;
            }
            
            $date = date("D M d, Y G:i");
            say_hello( 'World', $date );

    Result: 

                
              Hello, World
              Today is Fri May 10, 2019 4:00
    Basic Syntax

    PHP's syntax draws heavily from the C programming language and as such bears a strong resemblance to the syntax of other modern programming languages.

    Declaring a variable for example follows a familiar pattern if you have some experience with other languages: 

                
              $foo = 2;
              $bar = 2;
              $baz = $foo + $bar;
              echo "The variable \$baz = " . $baz;

    Result: 

                
              The variable $baz = 4

    Okay, you might be a little frustrated by these echo statements from the last two examples so here's a quick explanation: The echo is actually what is known as a language construct and we can pass parameters (i.e. $bar) to it and it will output them for us. The reason for the period ('.') between the two parameters is that we are concatenating, or combining, them into one argument which is passed to echo. We also use the backslash ('\') on $bar in the first parameter to tell php we want to output a literal ('$') symbol rather than retrieve the value of $bar inside our string. If we actual wanted to place $bar's value inside the string that would be known as string interpolation.

    And if that explanation made no sense to you please don't get too discouraged, this stuff takes a long time to get familiar with!

    Data Types

    PHP supports the following data types:

  • String
  • Boolean
  • Integer
  • Float
  • Array
  • Object
  • Null

    • If you're new to programming some of these data types like Arrays and objects may be unfamiliar to you. However you already know what ones like string, integer, and float are, at least in a basic sense. A string data type is basically just a collection of text (i.e. "Hello World"). Integer and float data types just hold whole numbers, and decimal numbers respectively. We'll go over some of the other data types in later sections.
